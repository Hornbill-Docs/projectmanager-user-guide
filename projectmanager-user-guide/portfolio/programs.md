# Programs
 Introduction A program is a set of related projects and activities, managed in a coordinated fashion and under a structure that allows for the delivery of business benefits. Each program is divided into these three areas: 
 * **Program Details** 
 * **Program Charts**
 * **Program Projects Overview** 

## Program Details

### Action Bar

* **Post** \- Add a post to the program activity stream
* **Projects** \- Search for and link projects to the program
* **Documents** \- Search and link documents from document manager to the program
* **Attachments** \- Upload attachments to the program
* **Members** \- Search and associate users and their roles to the program (Define the required roles via simple lists)
* **Business Benefits** \- Link business benefits (outcomes) to the program (Define your own business benefits via simple lists)
* **Boards** \- View, Add and Move the program on Board Manager boards

Once data is added to the program, an accordion section will appear on the program view to contain and provide access to the new data

### Program Details

* **Basic Details** \- Name, Description, Status, Sub Status, RAG Status and Progress of the program
* **Additional Information** \- Record specific program information
* **Dates** \- Record program start and end dates
* **Financial Information** \- Record currency and program budget information

Users with the **Form Designer** role, can add, amend, delete fields, set field validation and drag and drop to re-order.   

### Activity Stream

* Collaborate on program discussions with other program members
* Follow the program to receive real-time updates about the program on your newsfeed

### Program Overview

Visualize program RAG Status and Progress


### Projects

A bird's eye view of all projects in the program:

* Project Progress
* Project RAG Status
* Project Owner

* Drill into each project for more information
* Remove a project from the program

### Members

Search for, remove or view program members 

### Audit Trail

Review all changes to all program contents  
  
## Program Charts

Visualize and drill down into projects in the program

### Project RAG Status & Budgets

* View the number of projects in the program which are in each RAG status
* Drill down on each RAG chart to see the projects which make up this part of the chart
* Drill down further into each individual project

* View the number of projects in the program which are over or within their budgets
* Drill down on those within budget to see the specific projects, and what % they are within budget - drill down into any project as needed
* Drill down on those over budget to see the specific projects and what % they are over budget - drill down into any project as needed


### Project Gantt Chart

* Visualize all of the projects in the selected program
* Drill down further into each individual project

## Program Projects Overview

Review the progress of all projects in the program.
* Project Name, Manager, Progress, Status and Sub Status

* **Project Cost** \- Displays how each project is tracking against its budget in percentage terms i.e x % over or under the project budget
* **Project RAG Status** \- The current RAG status of each project in the program
* **Project RAG Tolerance Statuses** \- The RAG status for all defined RAG Tolerances on each project.

* The list can be filtered based on project status and searched on the project titles.
