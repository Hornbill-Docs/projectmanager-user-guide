# Portfolio Overview
The Portfolio provides visibility, access and supporting data relating to programs for your organization. The views include a list of Programs with high level details and progress and overviews of current program projects. Raise new programs by using the red **+** icon in the bottom right hand corner of the screen. 

## Programs

### List View

Programs are displayed in a list. The column order can be adjusted on a per user basis by selecting and dragging columns into position. Clicking the cog on the right hand side opens a popup window which allows each user to customize the view in terms of which columns are visible and are shown in what order, depending on their preference.  
  
#### Available Columns

* **Name**

The Name of the Program is a mandatory field for each program. This may be a descriptive name or in some cases program managers like to have code names for their programs

* **Program Manager**

This is the name of the person who is managing the program.

* **Start date**

Displays the start date for the program

* **End date**

Displays the end date for the program

* **Program Sponsor**

This is the name of the user assigned the program sponsor

* **Progress**

Displays the progress of the entire program (0-100%)

* **Status**

Displays the status of the entire program

* **Number of Projects**

Displays the number of projects that are included within the program

* **Last Modified By**

Presents the name of the person that last provided an update to the details of the program

* **Last Modified Date**

The date when the program was last updated

* **Sub Status**

Displays the sub status of the the entire program

* **Custom Fields**

Custom fields that have been added to a program. Long text fields (Custom B) are not available

#### Filters

* Search - Filter the displayed programs based on program name
* Status - Filter the displayed programs by program status
* Program Manager - Filter the displayed programs by program manager
* Program Sponsor - Filter the displayed programs by program sponsor

### Gantt View

Programs are displayed in a gantt chart which visualizes programs across the selected year. By default, the current year is selected on initial load. You can change the theme of the chart using the drop down menu on the top right hand side of the screen.  

#### Filters

* Status - Filter the displayed programs by program status
* Program Manager - Filter the displayed programs by program manager
* Program Sponsor - Filter the displayed programs by program sponsor
* Year - Filter the displayed programs by year

## Program Boards

Create and Manage Boards on your Portfolio view. Add Program Cards, as well as other Hornbill entity cards as required.
