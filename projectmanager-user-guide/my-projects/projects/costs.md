# Project Costs
Understanding and managing the costs associated with with a project is an important consideration. Associate estimated and actual costs to the project Review project costs by cost types and percentile splits. 

## Adding costs to a project

Use the **+** icon to add a new project cost

* **Description**: Add a description for the new project cost
* **Type**: Select which cost type the new project cost will be recorded against
* **Budget Code**: Select which budget code the new project cost will be recorded against
* **Currency**: The currency is set from the currency associated to the project in the project details section
* **Cost**: Provide a value for the project cost
* **Quantity**: Provide a quantity for the project cost

Use the **Add** button to add the project cost

* On adding the project cost, the costs will be added as estimated and actual costs and quantities

## Managing costs on a project

Edit or provide actual costs and quantities against project costs by clicking on the project cost you wish to edit from the list view

* **Description**: Update the description if required
* **Budget Code**: Update the budget code if required
* **Cost**: Update the Actual cost if different from the initial estimate
* **Quantity**: Update the Actual quantity if different from the initial estimate

## List View

View all project costs in list view, see current running costs against the projects budget.

* **Type**: Project Cost Type grouping
* **Description**: Description for the project cost
* **Budget Code**: Budget code for the project cost
* **Created By**: Who added the project cost
* **Estimated Cost**: The cost provided when the project cost was added
* **Estimated Quantity**: The quantity provided when the project cost was added
* **Actual Cost**: Current Actual Cost
* **Actual Quantity**: Current Actual Quantity
* **Total Cost**: The actual cost multiplied by the actual quantity
* Click on a project cost to review, or edit it's properties.
* Use the search option to find project costs by their description
* Use the Cost type option to filter the costs by a specific project cost type
