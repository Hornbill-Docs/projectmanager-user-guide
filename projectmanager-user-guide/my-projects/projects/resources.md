# Project Resources
Introduction View and Analyze all project resource contributions

### Resource Views

**List View** 

* View all Project Stakeholders
* See any currently assigned Active Tasks and % percentage progress through them
* Search Project Resources
* Drill into each Project resource to analyze their task and time contributions to the overall project

### Resource Analysis

Drill down onto a resources name to analyze their project contributions by task and time**Task Analysis** 

* View the Project Stakeholders current tasks, their due dates and their current % completion
* View how many Active Tasks the Project Stakeholder has
* View how many Tasks the Project Stakeholder has completed
* View in Percentage terms the Project Stakeholders contribution to the overall Project in Task terms, by their Active, their Completed and tasks of other Project Stakeholders

**Time Analysis - Recorded Time** 

* View a list of all the Recorded time for the Project Stakeholder on the Project, when it occurred and the time spent
* View the Project Stakeholders total time contribution to the project
* View the Overall resource time for the Project
* View in Percentage terms the Project Stakeholders contribution to the overall Project in recorded time terms against the overall Project recorded time.
